# betting-api

Bachelor's practice project.
Implementing API as described in given documentation.

- Creating a web page skeleton to put <iframe> element inside which sends requests to company's server. Server communicates with API integration
- Implementing XML based API.
- Implementing all required methods as described in the documentation
- Learning to work with tokens. Refreshing, updating...
- Testing API integration using postman and provided internal tools
- Additionally implementing a real-time functionality using "Pusher" API in order for user to see balance in real time.
